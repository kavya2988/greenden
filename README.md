# Greenden

A responsive, user-friendly e-commerce platform designed for purchasing indoor and outdoor plants. Built using HTML, CSS, and JavaScript, the site features a modern UI with plant catalogs, shopping carts, and scrolling animations. Ideal for a beginner frontend portfolio, this project showcases modern web design, CSS Flexbox/Grid, and responsive design techniques.

Plant E-commerce Website
This repository hosts a responsive plant selling website designed to provide a seamless shopping experience for plant enthusiasts. 
🚀 Features
Modern User Interface: Sleek, green-themed design focused on natural aesthetics.
Product Catalog: Showcases a variety of plants, succulents, and gardening tools.
Responsive Design: Fully compatible with mobile, tablet, and desktop devices.
Interactive Features: Includes scrolling animations and light/dark mode toggles.
User Flow: Browse, add to cart, and checkout functionality. 

 Technologies Used
Frontend: HTML5, CSS3 (Flexbox/Grid).
Styling: Tailwind CSS or Modern CSS.
Version Control: Git & GitHub Pages.
